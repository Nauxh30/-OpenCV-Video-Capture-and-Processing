# OpenCV Video Capture and Processing

## 📌 Overview
OpenCV provides powerful tools for capturing and processing live video through a webcam.  
Using Python and OpenCV, video frames can be captured, displayed, resized, and rotated in real time.  
Matplotlib is used to visualize the frames, while simple image processing operations improve flexibility and understanding of video manipulation techniques.

---

## 🎯 Aim
To develop a Python program using OpenCV for capturing images and displaying live video with operations such as resizing and rotating frames.

---

## 💻 Software Requirements
- Python 3.7+
- OpenCV
- Matplotlib
- Jupyter Notebook / VS Code / Anaconda

---

## 🛠️ Procedure

### i) Capture and Display an Image
1. Import the required libraries.
2. Access the webcam using OpenCV.
3. Capture a single frame from the webcam.
4. Save the captured frame as an image file.
5. Read and display the captured image using Matplotlib.

### ii) Display the Live Video
1. Open the webcam using OpenCV.
2. Continuously capture frames from the webcam.
3. Convert frames from BGR to RGB format.
4. Display the video frame by frame using Matplotlib.
5. Release the webcam after displaying the video.


---

### iii) Resize and Display the Video
1. Capture video frames from the webcam.
2. Resize each frame to a smaller resolution.
3. Convert the resized frame from BGR to RGB.
4. Display the resized video frames continuously.
5. Release the webcam after execution.

---

### iv) Rotate and Display the Video
1. Capture frames continuously from the webcam.
2. Rotate each frame clockwise using OpenCV.
3. Convert the rotated frame to RGB format.
4. Display the rotated video frames.
5. Release the webcam after completion.

## 📊 Output
- The webcam successfully captures and displays an image.
- Live video streaming is displayed frame by frame.
- Resized video appears in reduced dimensions.
- Rotated video frames are displayed correctly after rotation.
- Different video processing techniques are successfully visualized in real time.

## ✅ Result
Thus, image capture and live video processing operations such as displaying, resizing, and rotating video frames were successfully implemented using OpenCV in Python.  
The experiment demonstrates basic real-time video manipulation and processing techniques using webcam input.
